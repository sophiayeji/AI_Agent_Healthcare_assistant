# 🏥 Hospital Operations AI Assistant

An AI-powered healthcare analytics platform that transforms hospital operational CSV data into executive dashboards, AI-generated insights, operational risk detection, and actionable recommendations for hospital executives.

Built with Google AI Studio, Gemini, Google Agent Development Kit (ADK), TypeScript, React, and Vite, this application demonstrates how AI agents can support healthcare operations and executive decision-making.

---

# 🚀 Features

## Executive Dashboard

- AI-generated Executive Summary
- Operational Red Flags
- Root Cause Analysis
- AI-powered Recommendations
- Executive Report Export
- YTD & Monthly Performance Review

---

## Healthcare Analytics Dashboard

- Patient Demographics
- Staff Availability & Coverage
- Department Performance
- Length of Stay (LOS)
- Patient Satisfaction
- Monthly Operational Trends
- Executive KPI Cards

---

## Admin Portal

- Secure CSV Upload
- Multi-file Import
- Data Preview
- Sandbox Review
- Publish Workflow
- Live Dashboard Synchronization

---

# 🤖 AI Agent Capabilities

The Hospital Operations AI Agent automatically:

- Reviews uploaded hospital operational datasets
- Generates executive summaries
- Detects operational risks and bottlenecks
- Identifies performance trends
- Recommends actionable improvements
- Supports executive-level decision making
